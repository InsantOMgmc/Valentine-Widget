Справа внизу стандартный виджет с базовыми настройками — цвета, положение, скрипты появления и закрытия окна. Все базовые стили можно менять, с помощью переменных CSS и просто с помощью свойств. Вы можете выбрать любое положение виджета и расположение. Единственное, он не должен быть слишком большим, не должен быть по центру и не должен перекрывать много контента.

Требования к виджету:

  1. Виджет должен быть компактного размера и может размещаться у края экрана. Не выводим его как попап посередине.
  2. Можно экспериментировать с расположением и делать слева, справа. Снизу на всю ширину, но небольшой по высоте. Либо сверху.
  3. У виджета есть обязательные элементы, которые в коде отдельно выделены, их нужно оставить как есть, тк весь виджет работает на своем движке.
  4. В идеале картинки использовать SVG, чтобы размещать сразу в HTML, без внешних файлов.
  5. Все картинки должны быть свободными для коммерческого использования, ссылку на картинку обязательно нужно предоставить с результатом.

Чек-лист для проверки виджета:

  1. Виджет должен хорошо смотреться на мобильных 320px.
  2. Виджет должен быть достаточно компактным и не перекрывать сильно контент.
  3. На виджете должна быть анимация, чтобы привлекать внимание.
  4. На виджете обязательно должна быть кнопка закрытия

Техническая информация по коду

  1. Основной, родительской блок виджета — .wpshop-event.
  2. У него есть дата-атрибут data-u="#", в котором будет ссылка, это вставит потом сам движок
  3. Мы ему меняем только класс-модификатор wpshop-event--{{КОД_СОБЫТИЯ}}
  4. И для этого модификатора мы пишем все стили, которые меняем, например:
    .wpshop-event--new-year .wpshop-event__header { ... }
  5. {{КОД_СОБЫТИЯ}} — это название маркетингового события на латинице и со знаком минус,
    например, black-friday, 8-march, spring, new-year.
  6. У блока обязательно должен быть элемент .wpshop-event__close, он закрывает наш виджет по крестику, у него тоже нужно поменять {{КОД_СОБЫТИЯ}}.
  7. У виджета по умолчанию задана высота в CSS-переменной --wpshop-event-height, можно установить в auto.
